# 🏓 Pong Cottagecore 🌼

Un clásico reinventado: el mítico Pong, pero en versión *cottagecore*.  
Menos neón, más pradera.  
Menos arcade futurista, más picnic con flores silvestres.

Este proyecto es un juego hecho en **HTML, CSS y JavaScript **, generado con ayuda de Claude y luego ajustado a mano con amor campesino digital.

---

## 🌿 Descripción

Este es un remake del clásico Pong con estética *cottagecore*:

- 🎨 Colores suaves y naturales
- 🌸 Detalles florales y ambientación cálida
- 🧺 Estética artesanal y minimalista
- 🐑 Vibra tranquila pero competitiva

El objetivo es el mismo de siempre:
Evitar que la pelota pase tu paleta y hacer punto contra tu oponente.

Porque aunque estemos en una cabaña en el bosque, la rivalidad sigue intacta.

---

## 🎮 Cómo jugar

- A / Z: Jugador 1
- ⬆️⬇️ Jugador 2  
- La pelota rebota automáticamente

## 🛠️ Tecnologías usadas
